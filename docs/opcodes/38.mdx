## Notes

各命令は1バイトを占有しています。[PUSH](/#60)命令の場合、PUSHする必要のあるバイトがその後にエンコードされているため、その分コードサイズが大きくなります。

## Stack output

0. `size`: byte size of the [code](/about).

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Stack overflow.
