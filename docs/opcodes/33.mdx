## Stack output

0. `address`: コール元アカウントの20バイトのアドレス。最後にコールを行ったアカウントです（[delegate call](/#F4)を除く）。

## Example

| * | Input | Output |
|--:|------:|-------:|
| `1` | | `0xbe862ad9abfe6f22bcb087716c7d89a26051f74c` |

[Reproduce in playground](/playground?unit=Wei&codeType=Mnemonic&code='CALLER'_).

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Stack overflow.
